vs code shortcuts

Select current line	Ctrl + L  
select a word           ctrl + d  
Delete line	Ctrl + Shift + K  
Move line up/down	Alt + ↑ / ↓   
copy line up/down       Alt + shift + up /down  
Insert line below/above	Ctrl + Enter / Ctrl + Shift + Enter  
Toggle line comment	Ctrl + /  
Auto-complete / suggestions	Ctrl + Space  
Add cursor (click points)	Alt + Click   
add multicursor to all the existing words ctrl +f2  
Toggle terminal	`Ctrl + `  
New terminal	Ctrl + Shift + '  


for creating a virtual environment for python :use venv
source venv/bin/activate
